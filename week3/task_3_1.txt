Используем тематику предыдущего задания.

1. Необходимо реализовать возможность сохранять полученные сущности в виде XML документов. Структуру XML придумайте сами, но это не должен быть 1 элемент :) Обязательно, чтоб были и элементы, и аттрибуты!!! Например, для класса Person может быть такая схема XML документа:
<Person gender="m">
    <Name>
        <FirstName>Anton</FirstName>
        <LastName>Shabinskiy</LastName>
    </Name>
    <BirthDate>1987-05-02</BirthDate>
</Person>
Для каждой сущности, естесственно, формат XML будет свой.

2. Документы необходимо сохранять по папкам, каждый тип объектов в свою папку.

3. Реализовать возможность считывать XML файлы и из них создавать опять объекты соотв. классов.

4. Задание выполнить в виде .NET Core Console App (скачайте .NET Core, установите, разберитесь и создайте консольное приложение).
